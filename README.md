# Battleship

A terminal based battleship written in C.

## state

For the moment, we just have a 2 players mode.

## preriquisites

- C
- ncurses
- a terminal that supports colors

## wanna try it out?

```bash
cd path/to/cloned/directory

make test
```

Arrow keys, WASD keys to move the cursor.
Tab, enter to shoot.
